Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The paragraph element represents a paragraph or a block of text
  • The image element contains information about images that are displayed on the webpage
  • There are three types of list elements: unordered, ordered, and list items

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A border indicates how much space we want around the padding of an element
  • The height indicates the height of an element(in pixels:px)
  • The width indicates the width of an element(in pixels:px)
  • The font size sets a certain size to the font of an element(in pixels:px)

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git clone: clones a repository to your local machine
  • git add file-name: add one or more files to staging
  • git commit -a: commit any files you've added with git add
  • git pull: send a pull request to remote repository
